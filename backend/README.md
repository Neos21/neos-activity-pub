# NestJS Backend

## Installation

```bash
$ npm install
```

## Running The App (`http://localhost:3000/`)

```bash
# Development
$ npm run dev

# Production Mode
$ npm run build
$ npm start
```

## Test

```bash
# Unit Tests
$ npm test
# Watch
$ npm run test:watch
# Debug
$ npm run test:debug

# E2E Tests
$ npm run test:e2e
```
